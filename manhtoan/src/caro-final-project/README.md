# Confession App - August 2015

### Tool: install these app
Nodejs
Vagrant
Virtualbox

### Clone this repo - start rethinkdb
```sh
git clone https://github.com/zoontek/vagrant-rethinkdb
cd vagrant-rethinkdb
vagrant-up
```

### Install dependencies
```sh
npm install
```

### Start server
```sh
node index.js
```
